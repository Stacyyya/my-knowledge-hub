Astro로 만든 이 Knowledge Hub를 드디어 세상에 공개했다. 코딩 거의 안 해본 디자이너인데, Claude랑 대화하면서 설계부터 배포까지 끝냈다.

**시도한 것**

- 회사 깃랩이 아니라 내 **개인 깃헙(Stacyyya)** 에 올리고 싶었다. 그런데 이미 찍어둔 커밋들 작성자가 회사 이메일로 박혀 있어서, 전부 개인 신원으로 다시 썼다.
- `gh` CLI로 깃헙 로그인 → 저장소 생성 → push 까지.
- GitHub ↔ Vercel을 연결해서 **자동 배포**를 걸었다.

**배운 것**

- 커밋에는 "누가 만들었는지"가 이메일로 같이 박힌다. 공개 저장소라면 시작 전에 신원(이름·이메일)을 미리 정리해두는 게 깔끔하다.
- 깃헙 로그인은 토큰을 어딘가에 복붙하는 것보다 **브라우저 인증**이 안전하다.
- GitHub랑 Vercel을 한 번 연결해두니, 이제 글 하나 추가하고 `push`만 하면 사이트가 알아서 다시 배포된다. 이 TIL도 그렇게 올라간다.
